**Data Integration & SQL Automation System** 🚀
This project demonstrates a robust backend solution developed in Python for automated data extraction and structured storage. 
It reflects my core competencies in software engineering, API integration, and database management.

🛠️ **Tech Stack & Skills**
-**Language:** Python 3.x.
-**Libraries:** requests for REST API consumption.
-**Database:** SQLite for relational data persistence (SQL).
-**Architecture:** Clean Architecture principles with separation of concerns.
-**DevOps:** Environment isolation using venv and dependency management via requirements.txt.

**📋 Key Features**
-**Automated Extraction:** Consumes external REST APIs to retrieve real-time data.
-**SQL Persistence:** Dynamically creates and manages relational tables to store processed information.
-**Error Handling:** Implements try-except blocks to manage network latency and connection timeouts effectively.
-**Scalability:** Designed to be easily extended for different data sources or complex database schemas.

🚀 **How to Run**
-**Clone the repository.
-**Activate the virtual environment:** .\venv\Scripts\activate.
-**Install dependencies:** pip install -r requirements.txt.
-**Run the script:** python main.py.
